
# Gulcht's Dotfiles

just my tmux, neovim setup


## Setup Guide

Clone the project

```bash
  git clone https://github.com/gulcht/dotfiles.git dotfiles
```

Go to the project directory

```bash
  cd dotfiles
```

Make the installer executable and run it

```bash
  chmod +x install.sh
  ./install.sh
```

After installation, you can start:

```bash
  tmux
  nvim
```
